Python

Как уже было сказано язык python очень перспективный, соответственно у него много функций и модулей(который можно установить прямо в приложении PyCharm), вот в чем плюс Python. Не будем медлить начнем изучать базовые знания языка python. Как и в почти любом языке программирования в python есть переменные.Переменная в императивном программировании — поименованная, либо адресуемая иным способом область памяти, адрес которой можно использовать для осуществления доступа к данным. Данные, находящиеся в переменной, называются значением этой переменной. То есть, проще говоря переменная - это коробка, а то что в ней лежит - это значение переменной. В языке python переменной может любая буква или даже несколько слов состоящих из букв латинского алфавита, значением также может быть любая другая переменная, слово или число, к примеру:

a = 5
На примере видно то, что а - это переменная, 5 - это ее значение. Чтобы дать переменной значение, нужно между переменной и значением поставить знак равенства.

Следующая важная вещь в python это то, что в нем есть 4 основных типов данных это: int - число(пример: 5, 78), float - вещественное(пример: 5.4, 65.0) число, str - cтрока(пример: привет мир, при выедении на экран нужно писать в кавычках), boolean - логические переменые(их всего две: True, False).К примеру:

a = 45
b = 'Привет мир'
hello = 89.56
win = True

Команда print() выводит на экран то, что вы напишите в скобках после, помните, если вы пишите в скобках слова типа str, то не забывайте это вписывать их в кавычки:

Следующей командой будет условный оператор - if. Он нужен для того чтобы дать чему-то условие и указания, elif и else имеют такие же функции как и if, только используются для задачи других условий и указаний. И вот мы уже знаем почти все, чтобы запрограммировать калькулятор, осталось только изучить команду input(). Эта команда позволяет какому-то пользователю ввести что-либо. Перед самой командой пишется тип данных, который должен ввести пользователь: int(input('Введите любое число:'))

Теперь можно приступать к написанию простого калькулятора, который будет уметь только складывать и вычитать. Поверьте, это достаточно легко.

Вот код с объяснением:
what = input('Что делаем +, -: ') - #Создаем переменную и пишем input чтобы пользователь ввел действие(+, -)

a = float(input('Введите первое число: ')) - #Создаем переменную, передаем переменной float(input), чтобы пользоваель ввел первое число.

b = float(input('Введите второе число: ')) - #Создаем переменную, передаем переменной float(input), чтобы пользоваель ввел второе число.

if what == '+': - #Воспользуемся if чтобы совершить действие которое указал нам пользователь

c = a + b - #Здесь, переменная "c" хранит в себе cумму а и b

print("Результат:" + str(c)) - #Печатаем результат "с".

elif what == '-': - #Воспользуемся elif, чтобы считать другое действие которое ввел пользователь, в данном случае(-)

c = a - b - #Здесь, переменная "с" хранит себе разность а и b

print("Результат:" + str(c)) - #Также печатаем результат с помощью команды print()

else: - #Если же пользователь вводит (не - и не +), то мы выводим:

print('Неверное действие')

В некоторых местах вы могли заметить 4 точки перед строкой, вместо этих 4 точек вы у себя должны поставить 4 пробела. Если вы не отступите эти 4 пробела, то консоль выдадит ошибку и программа не сработает.
